# Ibrahim Almasri Portfolio

Static portfolio website.

## Deploy

1. Create a new public repository named exactly:

   `Ibrahimalmari.github.io`

   Replace `Ibrahimalmari` with your real GitHub username if different.

2. Upload all files from this folder to the repository root.

3. Go to the repository Pages settings.

4. Select:

   `Branch: main` and `Folder: /root`

5. Save.

Your website should be available at:

`https://Ibrahimalmari.github.io`

## Edit content

- Main page: `index.html`
- Styles: `style.css`
- Small interactions: `script.js`
- CV file: `files/IbrahimCVPlus26.pdf`
- Project pages: `projects/`

## Notes

This website is static, so it does not require PHP, Laravel, MySQL, Node.js, or a server-side runtime.
